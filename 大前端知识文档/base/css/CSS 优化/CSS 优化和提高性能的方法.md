# CSS 优化和提高性能的方法有哪些？

可以从 4 个方面来优化和提高性能：

## 1. 优化加载性能

* CSS 压缩；
* CSS 单一样式 --- 例如：选用 `margin: top 0 bottom 0` 而不是 `margin-top`、`margin-bottom`；
* 减少使用 `@import`，尽量使用 `<link>` 引入 CSS。

## 2. 优化选择器性能

* 避免使用通配符 `*`；
* 减少使用标签选择器，而是使用类选择器；
* 避免使用后代选择器；
* 尽量利用属性的继承；

## 3. 优化渲染性能

* 慎重使用 `float`、`position`；
* 尽量减少触发重排与重绘，尽量使用复合属性如：`translate`；
* 避免空规则集；
* 属性值为 0 时，省略单位；
* 属性为浮点数时，省略单位；
* 选择器嵌套，避免层级过深；

## 4. 提高可维护性、健壮性

* 抽离相同属性样式，整合为 class 使用，提高可维护性;；
* 样式与内容分离（将 CSS 代码定义到外部 CSS 文件中）。
