# HTML 语义化

语义化简单来说，就是根据内容的结构，选择合适的标签，便于开发者写出更优雅的代码同时让浏览器能够更好地解析网页。

## 语义化的目的

语义化的目的在于用正确的标签做正确的事。

## ✨语义化的好处

1. 提高代码可读性和可维护性，便于开发和维护；
2. 增强页面内容的结构化；
3. 利于 SEO；
4. 增强可访问性，利于屏幕阅读器定位和朗读；

## ✨常见的语义化标签

1. `<header>`
2. `<nav>`
3. `<section>`
4. `<main>`
5. `<article>`
6. `<aside>`
7. `<footer>`

❗HTML 语义化并不是一定要执行的标准！
