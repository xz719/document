# HTML 元素的分类

## 1. 按标签的闭合方式分

按标签的闭合方式分为：

* 双标签元素，如：`<p></p>`
* 单标签元素，如：`<br>`、`<img>`

## 💡2. 按元素的显示方式分

按元素的显示方式分为：

* 行内元素，例如：`<a>` `<b>` `<span>` `<img>` `<input>` `<select>` `<strong>`
* 块级元素，例如：`<div>` `<ul>` `<ol>` `<p>` `<h1>` `<h2>` `<h3>` `<h4>` `<h5>` `<h6>` `<dl>` `<dt>` `<dd>`

## 3. 按元素内容分

按标签的内容分为主内容类、表单类以及特殊内容类。
